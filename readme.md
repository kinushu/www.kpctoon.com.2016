KpcKara
===

## Description

a application template based by sinatra(padrino).

+ guard
    * scss compiling
    * auto reload
+ cap deploy (no impl)
+ administrator ('admin' branch)

## Requirement

## Usage

```
bundle exec guard -i
```

accces to:
http://localhost:9393

## Install

```
bundle install
```

## Setup

### Views

app/views

### Assets

scss files in 'assets/scss/' is compiled to 'public/css'
